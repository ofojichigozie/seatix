# Seatix Frontend

This folder contains the frontend web application for Seatix, built with React, Vite, Tailwind CSS, and TypeScript. It provides public event browsing, seat selection, user accounts, organizer tools, and admin views.

## Setup
1. Open a terminal in `seatix-frontend/`.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Run `npm run build` to create a production build.

## Useful Scripts
- `npm run dev` — start the development server.
- `npm run build` — build the application for production.
- `npm run preview` — preview the production build locally.
- `npm run lint` — check code with ESLint.
- `npm run format` — format source files with Prettier.
